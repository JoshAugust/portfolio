# mess.ai.ah - Version 2 Specification
# Generated: Feb 18, 2026

## Overview
mess.ai.ah is an AI-powered personal CRM and "Strategy Angel" designed to help users manage their professional network, discover strategic connections, and automate outreach. Version 2 introduces a hybrid architecture combining a React frontend with Python-based OSINT (Open Source Intelligence) tools for deeper digital footprint analysis.

## Core Value Proposition
- **"Network Savior"**: Turns a messy list of contacts into a strategic asset.
- **Automated Research**: uses AI and web scraping to enrich contact profiles.
- **Strategic Intelligence**: Identifies "high-leverage" connections based on user goals.
- **Privacy-Focused**: Uses local storage and client-side API keys where possible.

## Architecture & Tech Stack

### Frontend (User Interface)
- **Framework**: Vite + React (TypeScript)
- **Styling**: Tailwind CSS + Framer Motion (for "premium" feel and animations)
- **State Management**: React Context (`AppContext`)
- **Data Persistence**:
  - **Local Storage**: For API keys and temporary state.
  - **Firebase (Firestore/Auth)**: For persistent user profiles and contact data syncing.

### AI & Intelligence Layer
- **LLM Provider**: Google Gemini (via `@google/genai` SDK).
- **Service**: `geminiService.ts`
  - **Grounding**: Uses Gemini's built-in `{ googleSearch: {} }` tool for real-time web context in the frontend.
  - **Generative Features**:
    - **Strategy Analysis**: Identifies top contacts for specific goals.
    - **Deep Discovery**: Generates connection "talking points" and career fit scores.
    - **Event Scouting**: Finds relevant industry events.
    - **Email Drafting**: Auto-drafts outreach messages based on context.

### External Search / OSINT Layer (New in V2)
- **Search Provider**: DuckDuckGo (replaces Google Custom Search API to avoid quotas/blocks).
- **Frontend Search**: `services/searchService.ts`
  - Uses `duck-duck-scrape` (npm) for client-side web searches without API keys.
- **Backend/Scripting Tools (Python)**:
  - **Environment**: Python 3.13 (`.venv`)
  - **Key Libraries**: `duckduckgo-search`, `requests`, `beautifulsoup4`.
  - **Modules**:
    1.  **`digital_footprint.py`**:
        - **Purpose**: Verifies social presence across LinkedIn, Twitter, GitHub, Substack.
        - **Method**: DuckDuckGo site-specific search + HTTP verification (text matching).
        - **Output**: JSON report of verified URLs.
    2.  **`network_mapper.py`**:
        - **Purpose**: Maps relationships between a target and specific companies/topics.
        - **Method**: Heuristic search queries (e.g., `"Target Name" "Company"`, `"Target" "Conference"`).
    3.  **`content_analyzer.py`**:
        - **Purpose**: Analyzes a target's recent content (blogs, tweets) for topics and sentiment.
    4.  **`demo_*.py`**:
        - wrapper scripts to demonstrate the capabilities of the python modules (e.g. `demo_footprint.py`).

## Key Workflows

### 1. Onboarding & Goal Setting
- User defines their current role, industry, and a specific **"North Star Goal"** (e.g., "Raise Seed Funding").
- App configures the "Strategy Angel" context based on this goal.

### 2. Contact Management (The "Mess")
- **Import**: Supports CSV upload (e.g., LinkedIn export).
- **Enrichment**:
  - **"Deep Deep Discovery"**: The AI proactively scrapes web data (via Gemini Grounding or DuckDuckGo) to fill in missing details:
    - *Discovery Score*: How much info is public?
    - *Career Fit*: How relevant is this person to the user's goal?
  - **Avatars**: Generates AI avatars for contacts if photos are missing.

### 3. The "Priority Board"
- A Kanban-style view that sorts contacts by strategic value (e.g., "Sponsors", "Mentors", "Peers").
- **Smart Sorting**: The AI periodically re-evaluates contacts and suggests moving them to "Sponsor" status if they align with the user's goal.

### 4. Action Feed
- A dynamic "to-do" list generated by the AI:
  - "Draft email to [Name] about [News]"
  - "Congratulate [Name] on [Event]"
  - "Attend [Event] to meet [Name]"

## Data Structure (Types)
- **`Contact`**: `{ name, role, company, discoveryScore, careerFit, intelligence: { summary, talkingPoints } }`
- **`UserProfile`**: `{ name, goal, industry, interests, apiKeys }`
- **`FeedItem`**: `{ type: 'outreach' | 'event', rationale, tactics }`

## Setup Instructions

### Frontend
```bash
npm install
npm run dev
```
*Requires `VITE_FIREBASE_*` keys in `.env`.*

### Python Tools (headless)
```bash
python3 -m venv .venv
source .venv/bin/activate
pip install duckduckgo-search requests beautifulsoup4
python demo_footprint.py "Name" "Company"
```
